REXYGEN and Siemens PLCs 
===================================

The examples contained in this folder demonstrate the use of the REXYGEN System with Siemens PLCs (LOGO!, S7-300, S7-1200, etc.).

## Documentation ##

- [S7Drv - Communication driver for Siemens PLCs](https://www.rexygen.com/doc/PDF/ENGLISH/S7Drv_ENG.pdf)

- [REXYGEN Studio User Guide](https://www.rexygen.com/doc/PDF/ENGLISH/RexygenStudio_ENG.pdf)
- [Complete documentation of REXYGEN](http://www.rexygen.com/documentation-and-support)
